NovaNews Frontend
This is the React.js frontend for the NovaNews application. It provides the interface for browsing headlines, generating AI summaries, and managing a private library.

Prerequisites
Node.js installed

npm (Node Package Manager)

The backend server must be running on port 5000 for full functionality.

Setup and Installation
Clone the repository to your local machine.

Open a terminal in the project directory.

Install the dependencies:
npm install

Configuration
Open the file src/App.js.

Ensure the API_BASE_URL constant is set to http://localhost:5000.

Running the Application
Start the development server:
npm start

The application will open in the browser at http://localhost:3000.
